# AART-logo-font

A custom non-serif font for the AART logo with a stylized capital A and R. Narrower than [Space Grotesk](https://github.com/floriankarsten/space-grotesk/releases/tag/2) from which it was originally derived by a lot of AI to-ing and fro-ing.

The font name records are:

Family: AART
Subfamily: Regular / Bold
Full name: AART Regular / AART Bold
PostScript name: AART-Regular / AART-Bold

Install both TTFs and use font-family: 'AART' in your CSS. The specimen HTML is updated to match. It's a proper typeface now.
